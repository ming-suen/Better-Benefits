# BetterBenefits

BetterBenefits is a web application designed to help users make informed decisions about their health insurance plans. It allows users to estimate their annual medical costs based on their usage profile and compare different insurance plans to find the most cost-effective option.

## Features

-   **Profile Customization:** Users can select a standard usage profile (Low, Medium, High) or customize it with specific medical needs.
-   **Plan Comparison:** Users can add multiple insurance plans with details like Premium, Deductible, MOOP, Coinsurance, and Copays.
-   **Cost Estimation:** The application calculates the estimated annual cost for each plan based on the user's profile.
-   **Detailed Breakdown:** Users can view a detailed log of how the costs were calculated for each plan.

## Project Structure

-   `index.html`: The main HTML structure of the application.
-   `style.css`: The CSS styles for the application.
-   `script.js`: The JavaScript logic for the application.

## How to Run

Simply open `index.html` in your web browser.
